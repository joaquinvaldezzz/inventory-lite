# Inventory Lite

**Your Cross-Platform Inventory Management Solution**

Built with the [Ionic](https://ionicframework.com) framework and [Capacitor](https://capacitorjs.com), Inventory Lite is a native app designed for seamless inventory management. It leverages TypeScript for markup, [ESLint](https://eslint.org) for linting, [Prettier](https://prettier.io) for formatting, [shadcn/ui](https://ui.shadcn.com) for accessible components, and [Tailwind CSS](https://tailwindcss.com) for stylish, responsive design.

## Getting Started

### Prerequisites

Before you begin, ensure you have the following software installed on your machine:

- [Git](https://git-scm.com)
- [Node.js](https://nodejs.org/en)

#### Android Requirements

- [Android Studio](https://developer.android.com/studio)
- An Android SDK installation

#### iOS Requirements

- Xcode
- Xcode Command Line Tools
- [Homebrew](https://brew.sh)
- [CocoaPods](https://cocoapods.org)

### Installation

Follow these steps to set up Inventory Lite on your local device:

1. **Clone the Repository**

   Download or clone this repository by running:

   ```bash
   git clone https://github.com/joaquinvaldezzz/inventory-lite.git
   ```

2. **Navigate to the Project Directory**

   Change to the project directory:

   ```bash
   cd inventory-lite
   ```

3. **Install Project Dependencies**

   Install the necessary dependencies:

   ```bash
   npm install
   ```

4. **Start the Development Server**

   Run the development server in the background:

   - For Web:

     ```bash
     npm run dev
     ```

   - For Android:

     ```bash
     npm run dev:android
     ```

   - For iOS:

     ```bash
     npm run dev:ios
     ```

     You can view the app in your browser at [http://localhost:8100](http://localhost:8100).

### Additional Resources

To learn more about local development, testing, and deployment, refer to the official documentation for [Ionic](https://ionicframework.com) and [Capacitor](https://capacitorjs.com/docs).
