# Inventory Lite

This cross-platform native app is built with the [Ionic](https://ionicframework.com) framework and [Capacitor](https://capacitorjs.com). It uses TypeScript for markup, [ESLint](https://eslint.org) for linting, [Prettier](https://prettier.io) for formatting, [shadcn/ui](https://ui.shadcn.com) for accessible components, and [Tailwind CSS](https://tailwindcss.com) for styling.

## Getting started

### Prerequisites

Ensure the following software is installed on your machine:

- [Git](https://git-scm.com)
- [Node.js](https://nodejs.org/en)
- Android Requirements
  - [Android Studio](https://developer.android.com/studio)
  - An Android SDK installation
- iOS Requirements
  - Xcode
  - Xcode Command Line Tools
  - [Homebrew](https://brew.sh)
  - [CocoaPods](https://cocoapods.org)

### Installation

1. Download or clone this repository to your local device by running the following command:

   ```bash
   git clone https://github.com/joaquinvaldezzz/inventory-lite-ionic-capacitor.git
   ```

2. Navigate to the project directory using the following command:

   ```bash
   cd inventory-lite-ionic-capacitor
   ```

3. Install the project dependencies by running the following command:

   ```bash
   npm install
   ```

4. Start the development server using the following command and keep it running in the background:

   For Android:

   ```bash
   npm run dev:android
   ```

   For iOS:

   ```bash
   npm run dev:ios
   ```

   You may visit [http://localhost:8100](http://localhost:8100) to view the app in your browser.

To learn more about local development, testing, and deployment, refer to the [Ionic](https://ionicframework.com) and [Capacitor](https://capacitorjs.com/docs) documentation.
